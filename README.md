# bulletin_board

Backend - приложение на python, в котором реализовано управление объявлениями и комментариями к ним.

Для реализации приложения используются:
- Django;
- Django REST Framework;
- PostgreSQL;
- Docker.

Функции приложения:
- регистрация и авторизация пользователей, сброс и восстановление пароля через почту;
- управление объявлениями и отзывами, а именно чтение, создание, редактирование, удаление;
- поиск объявлений по названию через заголовок сайта;
- поддержка ролей "Пользователь" и "Админ". "Пользователь" управляет своими объявлениями, "Админ" управляет любыми объявлениями.

Приложение покрыто тестами.

Запуск проекта локально:
- клонируйте репозиторий https://github.com/matveinovikov1000/bulletin_board;
- создайте в корне проекта файл .env c переменными окружения по аналогии с файлом .env.example;
- создайте БД PostgreSQL;
- установите Docker https://docs.docker.com/engine/;
- установите зависимости;
- запустите миграции: python manage.py migrate

Зависимости, необходимые для работы проекта, описаны в pyproject.toml

Дополнительно приложение можно запустить контейнерах Docker: docker-compose up -d

Файл docker-compose.yaml описывает следующие сервисы:
- db: контейнер с БД PostgreSQL.
- app: контейнер с DRF-приложением.
